# This Area Of Code Is: Git Large File Storage (LFS) Configuration
# Explanation: This file tells Git how to handle large files (videos). Instead of 
# storing the actual video data in Git (which would break the 100MB limit), it 
# stores a text pointer and uploads the video to GitHub's LFS servers. This file 
# also documents the storage limits so you understand the constraints.
# In Other Words: This is the instruction manual telling Git "handle these big 
# video files specially - don't try to store them like normal text files."
# ================================================================================

# This Area Of Code Is: Video File Tracking Rule
# Explanation: Tracks all MP4 files using Git LFS. This replaces the actual video 
# data with a small text pointer in the repository, while the real file lives on 
# GitHub's LFS servers. Required because GitHub blocks files over 100MB.
# In Other Words: "Hey Git, these video files are too big for normal storage - 
# use the special big-file system instead."
*.mp4 filter=lfs diff=lfs merge=lfs -text
